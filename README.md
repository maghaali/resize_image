# resize_image
resizing images without touching quality

## How to use it?

Firsly pay attention that folders structure should be like:
-->CURRENT FOLDER
---->resize_image.py
---->images_folder_1
------>image1.jpg
------>image2.jpg
...
---->images_folder_2
------>image1.jpg
------>image2.jpg
...

Then run the following command:
```
python resize_image.py
```
